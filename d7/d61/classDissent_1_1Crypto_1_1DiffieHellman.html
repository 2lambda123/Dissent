Return a non-interactive zero-knowledge proof of a shared Diffie-Hellman secret.

A wants to prove to C that the shared secret between A and B was correctly generated by A. A then calls this method as:

private_key_A->ProveSharedSecret(public_key_B)

and sends the output to C

Verify a non-interactive zero-knowledge proof of a shared Diffie-Hellman secret.

A wants to prove to C that the shared secret between A and B was correctly generated by A. The verifier C then calls this method as:

public_key_A->VerifySharedSecret(public_key_B)

Returns:
QByteArray() if verification fails, otherwise returns the shared secret
