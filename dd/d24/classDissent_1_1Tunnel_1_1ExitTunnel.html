Detailed Description

This is the Dissent TCP tunnel "exit node." It reads TCP tunnel packets from the Dissent session and fowards them to the network address specified.

It broadcasts replies from the network connection non-anonymously to all members of the group.

!!!IMPORTANT!!! By serving as an exit node, the user running the exit node/remote tunnel gives up their anonymity.
