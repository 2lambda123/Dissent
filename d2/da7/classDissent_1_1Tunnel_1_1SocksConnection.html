Detailed Description

SocksConnection represents a single connection of the SOCKS server to a SOCKS client (most likely a user's Web browser).

SocksConnection encapsulates all of the SOCKS proxy negotiation logic and it also handles the packetization of data to be sent to the Dissent session.

SocksConnection supports the SOCKS v5 CONNECT and UDP ASSOCIATE commands.
