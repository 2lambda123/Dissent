Detailed Description

Object holding BlogDrop client ciphertext using pairing-based construction.

Every ciphertext element is an element of the pairing target group GT

The proof for a ciphertext of length k has the form: PoK{ a, y: ( C1 = e(prod_server_pks, t1)^a AND ... AND Ck = e(prod_server_pks, tk)^a AND A = g^a ) OR Y = g^y } where C1, ..., Ck are the k ciphertext elements, prod_server_pks is the product of server public keys, A is the client's public key, and Y is the author public key.
